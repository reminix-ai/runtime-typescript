# @reminix/runtime

Core Reminix Runtime for deploying AI agents via REST APIs.

## Installation

```bash
npm install @reminix/runtime
```

## Usage

```typescript
import { serve, BaseAdapter } from '@reminix/runtime';

// Serve wrapped agents
serve([wrappedAgent], { port: 8080 });
```
